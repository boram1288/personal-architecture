## 3. 전 DP를 관통하는 공통 게이트 (외부 확인 필요)

### 3.1 HV 파트 질의 (최우선)

- **H6 (SMMU 바인딩 제어 의미론)**: DP-02·03·11·12 4개 DP가 걸린 전역 최대 리스크. 최우선 단건 조기 질의 대상 (DP-03 G1). 미제공 시 DP-03 후보 B 폴백 + DP-12 재설계 + DP-11 상한 재평가 + DP-02 가정 변경의 연쇄.
- **H7 (pVM 초기 측정/attestation)**: DP-09 G1이 신설 요청, DP-02 2절 정본에 등재. DP-09·08·10과 DP-06 G2·DP-12 G5가 걸린 두 번째 리스크 묶음. H6과 같은 HV 세션에서 질의하되 폴백 트리거는 독립. 질의 시 측정 대상 granularity(pVM 최초 메모리 이미지 전체 vs 부트로더 영역만)를 명시 요청.
- H7 확인 시 연쇄: DP-10 G1·DP-08 G1/G2 완전 폐쇄 → DP-10·DP-08 골격→확정 전환 → DP-06 G2·DP-12 G5 완전 폐쇄.
- H7 미제공 시: DP-09 후보 C 폴백(초기 이미지만 측정) — payload-level 변조 차단 보장 불가를 전 문서에 한계로 연쇄 반영.

### 3.2 SRCX 협의

- TEE 신규 TA 자원 예산표 (DP-06 G5): DP-07 attestation TA(2단계 도입 시), DP-08 key TA(부팅 시 1회/workload, secure storage quota·boot storm rate limit), DP-09 측정값 보관·대조 TA(저빈도) 등록 완료.
- pVM발 TEE 호출 DoS 쿼터·레이트리밋의 집행 주체 (DP-06 G4, CONST-03 무회귀).

### 3.3 서비스 pVM TCB 편입 상한

- 현재 편입 사례: 로그 pVM 1건 (DP-11 G3, 최초 사례). 추가 편입은 통합 영향 평가 필요.
- DP-08 후보 B(key broker pVM)는 기준안에서 배제 — 도입하더라도 메타데이터·운영 보조 한정, 최종 unwrap/release는 TEE.

## 4. 검수에서 발견되어 결정문에 반영된 핵심 사항

- **H7 신설 (DP-09 검수)**: pKVM 초기 측정은 H1(lifecycle)도 H6(SMMU)도 아닌 별도 계약. 정본 보유는 DP-02(G5 정본 규율), DP-09는 요구 명세만 제공. H7이 "측정 보고에 자격 필요 ↔ 자격 발급에 측정값 필요"의 부트스트랩 순환을 끊는 메커니즘 — DP-04 부트스트랩 순서에 (i.5) 단계로 삽입됨.
- **DP-08 이중 의존 (CRITICAL)**: 키 방출 대상 판정은 "DP-09 측정값 → DP-10 measured identity → DP-08 방출"로 DP-10을 경유 — DP-08 골격은 양쪽 역반영에 걸림 (DP-08 G2).
- **P2 보안 게이트 우회 경계 (CRITICAL, DP-05 G2)**: 자체 키 관리·attestation을 가진 Secure OS가 DP-08 키 방출 게이트·DP-09 측정 체인을 우회하지 않을 의무를 conformance에 포함. 우회 허용 자산은 "프레임워크 보장 밖(P2 책임)".
- **패키지 anti-rollback (DP-05 G6·DP-09 G3)**: 구버전 패키지 + 구버전 sealed key 동시 롤백(과거에 유효했던 정당한 조합) 공격 차단 — manifest version·base image version 단조성을 TEE anti-rollback 카운터로 강제, DP-08 G4 key epoch와 단일 정책 통합.
- **TCB 범주 정밀화 (DP-05 G1, DP-09 G2)**: P1 베이스 이미지 = "per-domain 신뢰 베이스 + 측정 체인 루트"(전역 TCB 편입 아님). 공통 부트로더 = "전 pVM 공통 측정 체인 루트 코드"(EL2 초기 측정으로 보증, 전역 TCB 아님). 양쪽 모두 DP-01 5.4절(3) 서비스 pVM 편입 절차 비적용.
- **crypto-erase 한계 (DP-08)**: 분산 암호문 사본·백업·fleet key는 물리 제거 불가 — AT-4(공격자 모델 제외) 영역 귀속. 실행 중 pVM의 키/평문은 DP-02 자원 회수 + DP-04 G4 채널 해제와 연동(DP-08 G6).

## 6. 남은 작업

1. HV 파트에 H6·H7 질의 (동일 세션, 폴백 독립) — 회신 후 역반영 2단계(완전 폐쇄) 수행.
2. SRCX와 TA 예산표·DoS 쿼터 협의.
3. DP-02 G1: H1~H7 계약 명세 문서로의 승격 (시그니처·상태 전이·실패 원자성·오류 코드).
4. DP-03 1단계 PoC (저빈도 경로) — H1·H2 + H6 시그니처 1차 고정과 병행.
5. 완결 선언은 위 1번 회신과 역반영 2단계 완료 후.
